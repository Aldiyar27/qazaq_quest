# Qazaq Quest

Обновлённый demo-проект на ASP.NET Core MVC.

Что изменено:
- в каталоге оставлены только квесты по Астане
- без регистрации нельзя начать и проходить квест
- добавлены новые интересные маршруты по столице
- на карточках появились фото/иллюстрации и иконки
- улучшен визуальный стиль главной, каталога, деталей и прохождения
- добавлена более явная проверка задания с показом попыток
- сохранена GPS-проверка точки через геолокацию браузера

## Логика
1. Гость может смотреть каталог и описание квеста.
2. Для участия нужно зарегистрироваться или войти.
3. На каждой точке сначала проверяется геолокация.
4. После подтверждения GPS открывается форма ответа.
5. После верного ответа открывается следующая точка.

## Demo-admin
Если войти с email, содержащим `admin`, пользователю назначается роль `Admin`.

## Технологии
- ASP.NET Core MVC
- Session для демо-авторизации и прогресса
- HTML/CSS/JS без внешней базы данных
